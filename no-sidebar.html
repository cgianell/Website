Ongoing Project

Preview on what I'm working on right now

I am currently working on a game in Unity using C#. It is my first time making a game on unity and as such I intend to learn a lot about the Unity engine such as Fixed and Late Updates, Camera work, Scripting, Assets, Materials, and much more. In addition, I chose to make a game because not only will I improve my programming skills but it also allows me to practice artistic skills aswell through things like Blender (3d modelling and rigging), LMMS (music production), and Story Writing.

Work finished so far

I have gotten the hang of the player movement, most of the camera work and basic stage / prop management. Some of this was achieved through C# programming and others were basic features provided by Unity. However there is still a ton of work to be done.

Work to be done

I intend to make a rythym/rogue-like/shooter game similar to a mixture of Binding of Isaac, Osu and Call of Duty. To achieve that goal, I still need to work on many things such as the shooting mechanics, bullet mechanics, other projectiles, health mechanics, items, item synergies, abilities, and much more.
